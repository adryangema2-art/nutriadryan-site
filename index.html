NutriAdryan — Mostre seu produto

Tire uma foto, envie, e mostre o produto nutricional de forma profissional.

1) Tire a foto e transfira pro PC

  1. Tire a foto com seu celular (retrato ou quadrado funciona bem).
  2. Transfira pro computador via WhatsApp Web, cabo USB, Google Drive ou AirDrop.
  3. Se quiser, edite corte no app do celular antes de transferir.

2) Upload & pré-visualização

Escolha até 6 fotos. O site mostrará pré-visualização instantânea — nada é enviado a servidores, tudo roda no navegador.

Depois de satisfeito, você pode adicionar essas imagens ao repositório do GitHub (ou usar Google Drive/Imgur) para publicar o site com essas imagens permanentes.

Seu catálogo

Cada produto tem: foto, nome, descrição curta e botão "Ver/Comprar". Você pode editar o HTML para mudar preços e links.

Dica rápida

Fotos claras, fundo neutro e boa iluminação vendem muito melhor.

Como publicar (GitHub Pages) — passo a passo

  1. Crie uma conta no github.com (se não tiver).
  2. Crie um novo repositório: clique em New repository, dê um nome (ex: nutriadryan-site) e deixe público. Não precisa inicializar com README.
  3. No seu computador, crie uma pasta e coloque o arquivo index.html (este arquivo), e as imagens que quer usar.
  4. Faça o upload dos arquivos no repositório: vá no repositório no GitHub > clique em Add file → Upload files → arraste os arquivos → Commit.
  5. Vá em Settings → Pages (ou Settings → Pages and deployments dependendo da interface).
  6. Em Source, selecione a branch main (ou master) e a pasta / (root), clique em Save.
  7. Em alguns minutos seu site estará online em https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO/.

Como usar suas próprias fotos no site publicado

  1. Se você deixou as imagens no repositório do GitHub (passo de upload), substitua as referências locais no HTML por: <img src="NOME_DA_IMAGEM.jpg"> — usando exatamente o nome do arquivo que você enviou.
  2. Ou, se preferir hospedar imagens externamente (Google Drive, Imgur), pegue o link público e coloque no src da tag <img>.
  3. Faça commit das mudanças e o site será atualizado automaticamente.
