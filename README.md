Como publicar no GitHub Pages (resumo):
1. Crie conta em https://github.com se ainda não tem.
2. Crie um novo repositório (ex: nutriadryan-site) e mantenha público.
3. Faça upload do index.html e de quaisquer imagens que queira usar permanentemente (Add file → Upload files).
4. Vá em Settings → Pages e selecione a branch main/root e salve.
5. Em poucos minutos seu site estará em: https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO/

Como usar suas fotos:
- Para testar localmente: abra o index.html no navegador (duplo clique). O upload aqui é local e as imagens não saem do seu computador.
- Para publicar imagens permanentemente, faça upload dessas imagens no repositório do GitHub (ou use links públicos do Google Drive/Imgur) e atualize as tags <img src="..."> conforme o nome dos arquivos.
